\[P(5<T<7) = \int_{5}^{7} \frac{6t(10-t)}{1000} \, dt = \frac{6}{1000} \cdot \left[5t^2 - \frac{t^3}{3}\right]_5^7 = \frac{6}{1000} \cdot \frac{142}{3} = 0.284\]

\[ F(t) = \begin{cases} 0 & t \leq 0 \\ \frac{t^2(15-t)}{500} & 0 < t < 10 \\ 1 & t \geq 10 \end{cases} \]

x <- seq(0, 100, length.out = 300)
f <- x / 5000
plot(x, f, type = "l", lwd = 2, col = "blue",
     xlab = "x (grade)", ylab = "f(x)",
     main = "Density of Math Test Grades")

  1. \[E(X) = \int_{0}^{100} x \cdot \frac{x}{5000} \, dx = \frac{1}{5000} \cdot \left[\frac{x^3}{3}\right]_0^{100} = \frac{1{,}000{,}000}{15{,}000} = \frac{200}{3} = 66 \frac{2}{3}\].

  2. \[P(X \geq 90) = \int_{90}^{100} \frac{x}{5000} \, dx = \frac{1}{5000} \cdot \left[\frac{x^2}{2}\right]_{90}^{100} = \frac{10000 - 8100}{10000} = \frac{1900}{10000} = 0.19\]

  3. \[P(70 < X < 80) = \int_{70}^{80} \frac{x}{5000} \, dx = \frac{1}{5000} \cdot \left[\frac{x^2}{2}\right]_{70}^{80} = \frac{6400 - 4900}{10000} = \frac{1500}{10000} = 0.15\]

  4. This test is harder. The density increases linearly, so low grades are more common than high ones. With a mean of only 66.67 and just 19% scoring 90+, the distribution skews toward lower scores.

Problem 8

x <- seq(3, 6, length.out = 300)
f <- (4 + (x - 3)^2) / 21
plot(x, f, type = "l", lwd = 2, col = "blue",
     xlab = "x (minutes)", ylab = "f(x)",
     main = "Density of Race Completion Time")

  1. Slow times near 6 minutes are more common. The density is an upward-opening parabola: \(f(3) = 4/21 \approx 0.190\) while \(f(6) = 13/21 \approx 0.619\), so the density is much higher near 6 than near 3.

  2. \[P(X < 4) = \int_{3}^{4} \frac{4 + (x-3)^2}{21} \, dx = \frac{1}{21} \left[4x + \frac{(x-3)^3}{3}\right]_3^4 = \frac{1}{21}\left(16 + \frac{1}{3} - 12\right) = \frac{1}{21} \cdot \frac{13}{3} = \frac{13}{63} = 0.2063\]

  3. \[P(X > 5.5) = \int_{5.5}^{6} \frac{4 + (x-3)^2}{21} \, dx = \frac{1}{21}\left[4x + \frac{(x-3)^3}{3}\right]_{5.5}^{6} = \frac{1}{21}\left(24 + 9 - 22 - \frac{125}{24}\right) = \frac{1}{21}\left(11 - \frac{125}{24}\right) = \frac{1}{21} \cdot \frac{139}{24} = \frac{139}{504} = 0.2758\]

  4. \[F(x) = \int_3^x \frac{4 + (t-3)^2}{21}\,dt = \frac{1}{21}\left[4t + \frac{(t-3)^3}{3}\right]_3^x = \frac{1}{21}\left(4x + \frac{(x-3)^3}{3} - 12\right)\]
