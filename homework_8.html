S    <- 1000

set.seed(123)

pred_sd <- sqrt(sig^2 + sign^2)

x <- seq(-20, 60, length = 500)
plot(x, dnorm(x, mun, pred_sd), type = 'l', col = "blue", lwd = 2,
     main = "Posterior Predictive Distribution",
     xlab = "time-to-serve", ylab = "density")
lines(density(y_pred), col = "magenta", lwd = 2)

##       5%      95% 
## 10.76037 24.15775

The curves look very similar. The simulated deviates slightly near the peak at about 20 seconds. This deviation is due to Monte Carlo variability from using only S=1000.


The 90% prediction interval for a future time-to-serve is \((10.76037, 24.15775)\) seconds. This means we are 90% confident that Federer’s next single time-to-serve will fall in this range.

sim_max_fare <- function() {
  mu_draw <- rnorm(1, mean = mun, sd = sign)
  max(rnorm(20, mean = mu_draw, sd = sig))
}

max_fares <- replicate(1000, sim_max_fare())

hist(max_fares, breaks = 30, main = "Distribution of Maximum Fares",
     xlab = "maximum fare", col = "lightblue")
abline(v = 21, col = "red", lwd = 2)

The fare of \(\$21\) is relatively large compared to the distribution, it falls in the right tail. About \(4.2\%\) of the simulated max fares are at least \(\$21\).


About \(4.2\%\) of simulated maximum fares meet or exceed \(\$21\), meaning the observed maximum is somewhat uncommon under the model. This could suggest the Normal model with \(\sigma = 4\) may be slightly underestimating the spread of taxi fares, and a model with heavier tails or larger variance might be more appropriate.
